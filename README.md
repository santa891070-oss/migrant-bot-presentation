# Migrant Bot

Цифровая инфраструктура сопровождения трудовой миграции. Сервис для мигрантов, собственников жилья и работодателей. Доступен в мессенджере MAX.

Реализует пять пунктов раздела V Концепции государственной миграционной политики РФ на 2026–2030 годы (Указ Президента РФ № 738 от 15.10.2025).
